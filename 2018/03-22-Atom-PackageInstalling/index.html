Atom插件安装

Posted on 2018-03-22 | In tools , editor | Comments: | Views:
Atom的插件安装方式，包括从客户端安装，使用apm安装与使用git安装。

null方法一：使用客户端安装

使用客户端安装是最简单的方式。

点击Preference，在Settings中点击Install，然后输入需要安装的package的名字，搜索结束后点击需要安装的package栏目中的Install按钮即可。

null方法二：使用apm安装

apm(Atom Package Manager)是atom的包管理工具，可以方便的管理Atom的插件。
