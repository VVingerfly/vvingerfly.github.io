项目配置

配置项目的Header Search Paths和Library Search Paths，即包含目录和库目录。

双击项目打开testOpenCV.xcodeproj，点击Build Setting，再点击下方的Basic。在Header Search Paths和Library Search Paths分别添加OpenCV的相应路径。


具体路径可以通过brea info opencv获得。
